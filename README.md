# Cursor Replica

## Web app link

https://cursor-replica-sepia.vercel.app/

## Purpose of this repo
This is a replica of the cursor.com website created using HTML and CSS for *learning purposes*.
## Which sections were recreated

- Header
- Hero section
- Trusted-by
- Agents Section (Used image to create feature visual)
- Testimonials section
- Final CTA Section 
- Footer section

## Fonts and colors used
All the fonts and colors are from the original cursor.com website downloaded in assets folder.
